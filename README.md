# Comprehensive Automation Suite

## Overview
The Comprehensive Automation Suite is a Python-based desktop application designed to automate common system tasks.  
It provides multiple automation modules such as file organization, task scheduling, system monitoring, notifications, and logging through an easy-to-use graphical interface.

This project demonstrates practical implementation of automation tools using Python and showcases modular architecture, background task processing, and GUI integration.

---

## Features

• File Organizer  
Automatically organizes files into folders based on file type.

• Task Scheduler  
Schedules automation tasks to run at specific times or intervals.

• System Monitoring  
Monitors system performance such as CPU and memory usage.

• Email Notifications  
Sends alerts or notifications when specific events occur.

• Logging System  
Records application activities and errors for debugging and tracking.

• Graphical User Interface (GUI)  
User-friendly interface built using Python GUI libraries.

## Installation

1. Clone the repository


git clone https://github.com/likhithaavala9-star/comprehensive-automation-suite.git


2. Navigate to the project folder


cd comprehensive-automation-suite


3. Install dependencies


pip install -r requirements.txt


---

## Usage

Run the application using:


python main.py


The GUI will open and allow you to manage automation tasks.
